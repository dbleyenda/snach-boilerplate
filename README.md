# sNach Boilerplate

## Development

### Install globally
- Node: [https://nodejs.org/en/download/](https://nodejs.org/en/download/)
- Gulp: `sudo npm install --global gulp-cli`

### Install Dev packages
- `npm install`

### Dev server with live reload
- `npm run dev`

### Compile `build` folder for production:
- `npm run build`
